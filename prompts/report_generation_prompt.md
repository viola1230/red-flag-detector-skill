# Report Generation Prompt

根据结构化分析结果生成 Markdown 报告。

## 模板

```markdown
# 渣男鉴定报告

## 一句话结论

## 风险等级

## 类型标签

## 关键红旗

## 可能不是渣的部分

## 关系核心问题

## 你现在最该做什么

## 可复制话术

## 建议观察周期

## 姐妹分享版总结

## 风险提示
```

## 写作要求

- 可按用户 tone_preference 调整语气。
- 犀利吐槽可以有梗，但不要羞辱、定罪或煽动。
- 理性分析要多写证据链。
- 闺蜜口吻要给情绪支持，但建议必须具体。
- 温柔提醒要降低攻击性。
- 报告体要结构化、克制。

## 必须包含

- 证据来自哪里。
- 哪些部分是推测。
- 哪些部分可能只是沟通差或节奏不一致。
- 下一步可执行建议。
- 一段可复制边界话术。
- 安全提示。
